Tujuan dibuatnya web Re:Index


Hallo, jika kamu masih kepo tentang kami, mungkin jawaban-jawaban disini bisa membantu kamu , mungkin isi dari jawaban pertanyaan disini tidak spesifik tapi masih bisa telan sedikit lah :D

Apa itu Re:Index?
Re:Index adalah web yang berisi informasi teknologi atau Inovasi bahkan Tutorial mengenai programming atau coding yang bersangkutan tentang teknologi entah itu dalam bentuk digital atau material.

Apakah Re:Index rutin memposting artikel?
Mungkin jawaban disini tidak, karena pada awalnya web ini hanya sebatas project latihan penulis ketika mempelajari web programming, atau sekedar keisengan saja.

Apakah admin programmer?
Saya pribadi bukan disebut programmer karena saya juga masih newbie yang sedang belajar!

Apakah jikalau kita tidak memiliki background IT bisa coding?
Coding tidak hanya dilakukan oleh programmer, setiap orang entah itu memiliki background IT atau tidak, berhak belajar pemrogaman. Pemrogaman itu seperti Matematika, ada masalah dan ada solusi.
